```python
print("Uts Susulan")

xangka=0
while True:
    xbilangan = int(input("Masukkan Bilangan : "))
    if (xangka < xbilangan):
        xangka=xbilangan
    if (xbilangan == 0):
        break

print("Bilangan terbesar adalah: ",xangka)
print()
```
